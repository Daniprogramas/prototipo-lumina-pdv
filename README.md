# prototipo-lumina-pdv

Protótipo do sistema Lumina PDV, desenvolvido para a disciplina de Engenharia de Software da UFRN.

## Objetivo

Desenvolver um sistema de Ponto de Venda (PDV) acessível para pessoas com baixa visão, com foco em:

- Login com controle de acesso
- Cadastro de operadores
- Cadastro de produtos
- Registro de vendas
- Controle de estoque
- Recursos de acessibilidade

## Tecnologias

- Figma (Protótipo)
- Python
- Django
- HTML
- CSS
- JavaScript
